Projeto de desenvolvimento de um workflow de SPA utilizando gulp.

O projeto foi desenvolvido durante o curso "Web Moderno com JavaScript 2020 COMPLETO + Projetos" através da Udemy, ministrado pela instituição de ensino Cod3r.

Foi possível desenvolver workflows automatizado em paralelo ou em serie, e utilizar ferramentas como sass, babel, concatenação de códigos em um único arquivo e minificação do código.
